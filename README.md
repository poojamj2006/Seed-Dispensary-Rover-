# Seed Dispensary Rover

## Overview

The Seed Dispensary Rover is an Arduino-based autonomous agricultural robot designed to automate the seed sowing process. The rover moves through the field, dispenses seeds at regular intervals, and detects obstacles to avoid collisions. This project aims to reduce manual labor, improve farming efficiency, and support smart agriculture practices.

## Problem Statement

Traditional seed sowing methods require significant human effort and often result in uneven seed distribution and seed wastage. Farmers need a cost-effective and automated solution to improve productivity and accuracy in seed placement.

## Proposed Solution

The Seed Dispensary Rover automates seed sowing using an Arduino Uno, DC motors, an ultrasonic sensor, and an SG90 servo motor. The rover navigates through the field, detects obstacles, and dispenses seeds automatically.

* Battery Pack

## Working Principle

1. The rover moves forward through the field.
2. The ultrasonic sensor continuously monitors for obstacles.
3. If an obstacle is detected, the rover changes direction.
4. The SG90 servo motor rotates and dispenses seeds.
5. Seeds are placed at regular intervals while the rover moves.
